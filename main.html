Bot Overview

NineAsh is a custom Discord moderation & utility bot crafted with Python and discord.py. The core focus here is the command‑handler architecture that powers every feature – from anti‑spam to music playback.

Timeline: Jun 2025 – Present

Key Features

Moderation System

  • ✅ Custom profanity filter (English + Hindi).
  • ✅ Anti‑spam timeout (5+ messages in < 5 s).
  • ✅ Mod‑mail bridge between DMs & staff channel.
  • ✅ Role‑restricted command groups.

Interactive Chat

  • 💬 Friendly replies for casual greetings.
  • ❓ General‑knowledge Q&A.
  • 🎤 Joke & small‑talk fallback.

Search Utilities

  • wiki <query> – instant Wikipedia summaries.
  • search <query> / !search – DuckDuckGo web search.

Music & Media

  • ▶️ Queue, play & control tracks in voice channels.
  • 👋 Custom audio/video welcomes.

Command Reference

General

  • !help – display all commands.
  • !ping – latency check.

Moderation

  • !kick <user>, !ban <user>
  • !timeout <user> <secs>
  • !triggers – list auto‑responses.
  • !reply <user> <msg> – respond to mod‑mail.

Music

  • !play <url/search>
  • !skip, !pause, !queue

Technical Highlights

The bot is built on an event‑driven architecture. A central dispatcher maps Message and Interaction events to modular cogs, enabling:

  • 🔌 Hot‑reload of cog modules.
  • 📑 Structured logging to JSON.
  • 🗄️ SQLite persistence for spam counters.
  • 🛡️ Graceful error boundaries with user‑friendly replies.
